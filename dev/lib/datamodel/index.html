Data Model API

The genser data model is the intermediate representation between the application data and types serializable by a serialization format.

Metadata Types

Genser.Tag — Type
Genser.Tag{tag}

The type of genser tags.

Tags provide additional semantic metadata about a genser type. They are particular useful for string values because some (de)serializers are capable of including additional metadata in their output.

Each tag is a singleton of a unique data type.

Macros

Genser.@fieldtype — Macro
Genser.@fieldtype(type, fieldkey, gensertype)

Override the genser type for a struct field.

using Genser

struct Person
    id::Base.UUID
    name::String
end

# Always convert the UUID to a string
@fieldtype Persion :id GenserString
source
Genser.@fieldencoding — Macro
Genser.@fieldencoding(type, fieldkey, encodingkey)

Set the string encoding for a binary value.

using Genser

struct DataContainer
    contents::Vector{UInt8}
end

# Encode as base64 when serialized as a string
@fieldencoding DataContainer :contents :base64
